Краткое содержание файлов репозитория libdrome
===========================================

index.html
- Точка входа для браузера. Настраивает canvas (`id="game"`), управляет DPR, экспортирует `window.__GAME_CANVAS` и `window.__GAME_CTX`, читает `liblist` и динамически подключает модули как type=module.

liblist
- Текстовый файл со списком JS-файлов в папке `lib/`, по одному пути на строку. Используется `index.html` для загрузки.

lib/Game.js
- Исполнитель: создаёт `MapTiler`, `MapGen`, карту, загружает её в tiler, создаёт `Camera`, запускает RAF-цикл, вызывает `update` и `draw` для тайлера и сущностей.

lib/MapTiler.js
- Управляет структурой карты: `createEmptyMap`, `loadMap`, `getTile`, `setTile`, `registerVariant`. Рендерит видимую область в `draw(ctx, viewport)`; простые flat/full варианты с отрисовкой цвета и простыми эффектами.

lib/MapGen.js
- Генератор карт: заполняет базовый слой травой, выполняет небольшой flood-fill для воды, размещает камни и бревна, вызывает `EntitySpawner` для спавна слаймов и добавляет их в `map.entities`.

lib/Camera.js
- Управление позицией и зумом камеры. Поддерживает колесо мыши (плавный зум с сохранением точки под курсором), клавиши WASD/стрелки и Shift для ускорения. Возвращает `viewport` для отрисовщика.

lib/Entity.js
- Базовый класс сущностей: хранит позицию в пикселях, размеры в тайлах, флаг коллизии. Реализует `move(dx,dy,tiler,entities)` с попытками скольжения и мелких шагов, `_canMoveTo` для тайловой и сущностной коллизии, `draw` — отладочный прямоугольник.

lib/EntitySlime.js
- Подкласс `Entity`, рисует круглый «слайм» с глазами, учитывает `facing`. Вспомогательная функция `shadeColor` для отрисовки обводки.

lib/EntitySlimeSample.js
- Поведение блуждающего слайма: наследует `EntitySlime`, генерирует случайные вектора скорости, реализует `update` для смены направления и передвижения с использованием `move`.

lib/EntitySpawner.js
- Логика спавна: ищет 3x3 области базового слоя, пригодные для размещения 2x2 слаймов, создаёт `EntitySlimeSample` и проверяет столкновения, выполняет «nudge» при установке.

lib/CubeSample.js
- Вспомогательный пример: класс 3D-куба с вращением и простой перспективной проекцией для отрисовки на 2D-canvas. Не влияет на основную логику тайлового мира.

lib/Sample.js
- Пустой файл (зарезервировано для примеров).

AI_CONTEXT/
- `README.md` — этот контекстный файл (руководство для ИИ‑преемников).
- `SUMMARY.md` — этот файл: краткие заметки по всем файлам.

Замечания
- Репозиторий небольшой и хорошо документирован комментариями в коде. Чтобы продолжить, рекомендую:
  - протестировать локально через `python3 -m http.server` и открыть `index.html`;
  - при внесении правок сначала обновить `liblist`, если меняется расположение файлов;
  - поддерживать `AI_CONTEXT` в актуальном состоянии.
