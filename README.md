# libdrome
Библиотека-ядро для сгенерированных проектов, использующая разделение кода на отдельные js-файлы, для ограничения доступа ИИ к редактированию кода.

### Ссылка Pages
https://offlinejust.github.io/libdrome/index.html

В папке lib хранятся js-файлы: модули и исполнители. Модули должны содержать только классы, методы и иные шаблонные конструкции. Исполнители должны содержать всё остальное, кроме классов, методов и шаблонных конструкций. Всё содержимое папки lib (кроме данной текстовой заметки) перечислено в файле liblist, с указанием пути. Файл liblist не должен содержать никаких лишних символов, кроме указания пути к файлам.
